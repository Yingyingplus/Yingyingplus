华为云函数搭建OneDrive网盘

发表于2022-03-28|更新于2022-03-28|好玩
|阅读量:

网上的教程大多关于腾讯云函数，很少有华为云的。直到我搞完才知道，华为云有多拉…
用腾讯云吧，快逃！

简介

函数工作流（FunctionGraph）是一项基于事件驱动的函数托管计算服务。通过函数工作流，只需编写业务函数代码并设置运行的条件，无需配置和管理服务器等基础设施，函数以弹性、免运维、高可靠的方式运行。此外，按函数实际执行资源计费，不执行不产生费用。

准备

  • 实名认证后的华为云账号；

  • OneDrive 账号，个人账号容量低，推荐申请 365 开发者账号 E5 或者找个 E5 子账号或者国内的世纪互联；

  • 下载 OneManager 压缩包（zip 格式的）。

配置

这里有 OneManager 的 中文文档 ，可以参照阅读。

  1. 登录 华为云 ，鼠标移到右上角个人信息那里，下拉菜单中点击我的凭证；
  2. 点击下面的访问密钥，点击新增访问密钥，描述随便填，会有一个手机验证码验证；
  3. 下载访问密钥，注意保存好文件，后面会用到；
  4. 直接跳转到 函数工作流 ，点击右上角创建函数；
  5. FunctionGraph 版本选择 v1，函数名称随便填；

  1. 选择运行时语言为PHP7.3，点上传ZIP文件，选择文件；

注意，这里的 zip 文件不能是直接下载的，要先解压，跳过外层的文件夹，选中里所有的文件直接压缩。

  1. 创建触发器。首先点分组右边的创建分组，点击右上角的创建分组，名称就用它推荐的就好啦。然后回到这个页面，类型选择 API 网关服务，名称随便填，安全认证选None，后端超时（毫秒）将5000改成30000。

  1. 复制调用 URL 到浏览器打开，点击安装。

设置OneManager

  1. 选择语言，设置密码，根据提示输入密钥 ID，进入空白的 OneManager 页面。这就是你的网址了。
  2. 点击左上角管理，设置，添加盘，根据自己的 OneDrive 账号类型 选择，登录微软账号；
  3. 自己美化吧。

注意，如果没有弹出登录框，是因为使用 Edge 浏览器，自动登录了当前账号，如果需要登录别的账号，请将当前浏览器账号退出，再使用别的账号登录。

接下来是 bug 环节，上传文件或者打开文件夹等操作有时会报错，有时又可以。API 测试没问题，我猜测是环境变量的问题，懒得折腾了。

自定义域名

没钱买域名，没有尝试，类似教程很多，自己找吧。
